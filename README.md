# Card Game

Build a Card Game with React Native.

## Running the App

To run, run the following commands. after execute the 'npm start', follow the instraction on the terminal.

```bash
  npm install --global expo-cli
  npm install
  npm start
```

Can run the app on iOS or Andorid devices using following commands

iOS :

```base
npm run ios
```

Android :

```base
npm run android
```

[More Details for Expo CLI](https://docs.expo.dev/get-started/installation/#1-expo-cli)
